# Jinn ImageViewer

一个用 Rust 编写的 Windows 图片查看器，支持单图/双图模式、缩放、适应窗口、快捷键自定义等功能。

## 功能特性
- **单图/双图模式**：支持单张图片查看和双列并排查看。
- **缩放控制**：鼠标滚轮缩放，支持自定义缩放步长。
- **适应窗口**：一键适应窗口大小（鼠标右键或快捷键）。
- **快捷键自定义**：可自定义打开文件夹、切换图片、删除图片等操作的快捷键。
- **状态栏显示**：显示当前图片名称、索引、缩放比例等信息。
- **关于窗口**：查看作者信息和项目版本，快速访问 GitHub。

## 安装
1. 下载最新版本的 `jinn_imageviewer.exe`。
2. 双击运行即可。

## 使用说明
1. **打开图片**：点击菜单栏“文件”->“打开文件夹”，选择包含图片的文件夹。
2. **切换图片**：使用快捷键（如箭头键）或菜单栏“视图”->“上一张/下一张”。
3. **缩放图片**：滚动鼠标滚轮。
4. **适应窗口**：点击鼠标右键或在菜单栏“视图”->“适应窗口”。
5. **自定义快捷键**：点击菜单栏“设置”->“自定义快捷键”，可修改各操作的快捷键。

## 快捷键默认设置
| 操作                | 快捷键       |
|---------------------|-------------|
| 打开文件夹          | O           |
| 上一张图片          | ← / ↑       |
| 下一张图片          | → / ↓       |
| 删除当前图片        | Delete      |
| 切换双图模式        | F2          |
| 复制当前图片        | Num1        |
| 复制右侧图片        | Num2        |
| 适应窗口            | V / 鼠标右键 |
| 自定义快捷键        | 在设置中修改 |
